回覧チェックソフトについて

１　回覧チェックソフトのインストール方法
Installというファイルを共有フォルダにおいて、ユーザーから自分のPCのデスクトップなどにコピーしてもらう。
・ダブルクリックで実行
・グループセッションに登録してあるユーザ名になっているか確認し、ちがっていたら修正する。
（ユーザー名は原則として、県のメールアドレス名に一致させていますが、違っている方もおりますので事前に
配付受けたユーザー名をご確認ください）
・「設定保存＆未読チェック開始」をクリック
・未読チェック動作が開始されれば、画面の下のほうに、次回の実行時刻が表示されます。

以上の操作で、グループセッションと設定ソフトNotifySetのショートカットがデスクトップに作られます。
また、１時間ごとチェック（間隔は変更可能）して、未読があればポップアップを表示します。

２　アンインストール方法
・UnistallというファイルをCドライブの「NotifyGs」というフォルダからデスクトップにコピーして
実行して、画面に出てきた「チェック停止＆設定の削除」をクリックしてください。


３　グループウエアの表示方法
・gsessionというショートカットをダブルクリックします。
・タスクトレイに未読のポップアップが表示された場合、その表示をクリックするとグループセッションが表示されます。


４　未読チェックの設定変更の方法
NotifySetというショートカットを実行すると設定場面が表示されます。設定変更が必要なときお使いください。
