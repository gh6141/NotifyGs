１　回覧チェックソフトのインストール方法
NInstallというファイルを共有フォルダにおいて、ユーザーから自分のPCのデスクトップなどにコピーしてもらう。
・ダブルクリックで実行
・グループセッションに登録してあるユーザ名になっているか確認し、ちがっていたら修正する。
（ユーザー名は原則として、県のメールアドレス名に一致させていますが、違っている方もおりますので事前に
配付受けたユーザー名をご確認ください）
・「設定保存＆未読チェック開始」をクリック

以上の操作で、グループセッションのショートカットがデスクトップに作られます。
また、１時間ごとチェックして、未読があればポップアップを表示します。

２　アンインストール方法
・NUnistallというファイルをCドライブの「NotifyGs」というフォルダからデスクトップにコピーして
実行して、画面に出てきた「チェック停止＆設定の削除」をクリックしてください。
